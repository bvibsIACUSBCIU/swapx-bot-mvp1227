## 🎯 使用流程

### 1️⃣ 导入钱包
- 点击左侧"钱包管理"
- 输入私钥（格式：0x开头的64位十六进制）
- 点击"导入钱包"按钮

### 2️⃣ 选择策略
- 点击"策略设置"
- 选择"定投策略"或"网格策略"
- 配置策略参数：
  - **定投策略**：买入金额、执行间隔、总次数
  - **网格策略**：网格数量、价格区间、单网格金额

### 3️⃣ 启动机器人
- 点击"控制台"
- 点击"启动"按钮
- 机器人自动执行交易

### 4️⃣ 查看日志
- 在"控制台"或"运行日志"查看实时日志
- 可导出日志文件

## ⚠️ 安全提示

1. **私钥安全**
   - 私钥仅保存在浏览器localStorage
   - 清除浏览器数据会删除私钥
   - 建议使用测试钱包

2. **交易风险**
   - 先在测试网测试
   - 小额资金开始
   - 设置合理的止损

3. **代码审查**
   - 使用前请审查代码
   - 确认合约地址正确
   - 检查RPC节点可用性

## 🐛 常见问题

### Q1: 安装依赖失败？
```bash
# 清除缓存
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

### Q2: 无法连接区块链？
- 检查RPC节点地址是否正确
- 确认网络连接正常
- 尝试更换RPC节点

### Q3: 交易失败？
- 检查钱包余额是否充足
- 确认代币合约地址正确
- 检查是否已授权代币
- 查看控制台错误信息

### Q4: 页面样式错误？
```bash
# 重新安装antd
npm install antd@latest
```
